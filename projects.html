    Slow Loris (DoS)

    Slow Loris is a DoS method, which anyone can use even with a low bandwith. It is making numerous connections to a webserver, then sending data every 10 / 15 seconds, just when the webserver would close the connection, so that it would not close. I made 2 versions of this script. One you can use with your own IP, but that is not, recommended. The second script, is using TOR Proxies, so your IP won't be visible. Both of the scripts are written using Python.
